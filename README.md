# SkyNav ✈️

A mobile-first Progressive Web App (PWA) for travelers to navigate airports 
and optimize layover time.

## Features
- Live layover countdown timer
- Airport navigation with gate-to-gate routing
- Amenity finder (lounges, cafes, washrooms, restaurants)
- Flight status tracking
- Works offline (PWA)
- 8 airports supported: LHR, DEL, DXB, BOM, JFK, KUL, ZRH, LAX

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started
```bash
npm install
npm run dev
```

## Deploy
Built for Vercel. Add `VITE_AVIATIONSTACK_API_KEY` in environment variables.
